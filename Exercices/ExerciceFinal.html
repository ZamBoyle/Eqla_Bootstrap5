Exercice Final - Bootstrap

Bonjour chers étudiants !

Vous l'attendiez, vous en rêviez, le voici !

L'exercice final de Bootstrap !

  • Vous allez devoir créer un site web en utilisant Bootstrap 5.3.
  • Vous devrez me rendre cet exercice final pour le 29 février 2024 au plus tard.
  • Vous avez la théorie et les exercices pour vous aider.
  • Vous enverrez votre site sur GitHub Pages (voir le point 3).

1. Objectifs

Créer un site web en utilisant Bootstrap.
Choisissez un thème (entreprise, club, recettes de cuisine, etc) et faites-en un site web avec les fonctionnalités suivantes :

  • Utiliser Boostrap 5.3
  • Une barre de navigation avec un logo et 4 liens: Accueil, Produits, A propos, Contacts.
  • La barre de navigation deviendra un menu burger si on atteint un certain breakpoint que vous aurez défini pour un smartphone.
  • La barre de navigation aura un lien d'évitement pour les personnes malvoyantes et permettra d'aller directement au contenu.
  • Toutes les pages doivent avoir le même menu de navigation.
  • N'oubliez pas d'activer dans le menu la page sur laquelle vous êtes.
  • Une title différent pour chaque page.
  • Différents niveaux titres: h1, h2, etc.
  • Tous les titres h1 doivent être centrés.
  • N'oubliez pas que toutes les images doivent être responsives.
  • Un footer avec le symbole © suivit de l'année et du nom de votre site à gauche. Par exemple: © 2023 - Cupcakes Crafters
  • Quand vous utiliserez du HTML posez-vous la question si nous n'avons pas vu quelque chose en rapport en Bootstrap.

2. Descriptifs des pages

2.1 Accueil

- Le titre1 sera centré.
- Une page d'accueil avec une image après le titre1.
- Mettez un message d'accueil attrayant à vos visiteurs.

2.2 Produits

- Le titre1 sera centré.
- Une page avec une liste de produits (au moins 10) avec une image, un titre, une description.
- Vous pourriez utiliser Les cards de Bootstrap présenter votre produit.

2.3 À propos

- Le titre1 sera centré.
- Une page avec une image et un texte qui raconte l'histoire de votre site.

2.4 Contacts

- Une page avec les coordonnées de votre site (adresse, téléphone, email, etc).
- De plus, ajoutez un formulaire de contacts qui aura:

  • Un champ Nom.
  • Un champ Prénom.
  • Un champ Email.
  • Un champ Message.
  • Un champ Bouton Envoyer.

3. GitHub Pages

Vous devez mettre votre site sur GitHub Pages.
Vous allez donc devoir créer un nouveau repository sur GitHub et y mettre votre site.
Suivez cette procédure qui est disponible dans le cours.
