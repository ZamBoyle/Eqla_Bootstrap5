Bootstrap - navbar

Introduction

Je vais vous présenter des exemples de navbar différents que vous pouvez créer avec Bootstrap.

Je vais partir de ce code que je vais copier-coller pour les autres exemples et je vais modifier les classes pour obtenir les autres exemples.

    <div class="container-fluid">
        <nav class="navbar navbar-expand-sm navbar-light bg-light">
            <a class="navbar-brand" href="#">Navbar</a>
            <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav0"
                aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
                <span class="navbar-toggler-icon"></span>
            </button>
            <div class="collapse navbar-collapse" id="navbarNav0">
                <ul class="navbar-nav">
                    <li class="nav-item">
                        <a class="nav-link" href="#">Accueil</a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" href="#">Fonctionalité</a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" href="#">Prix</a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" href="#" tabindex="-1">Contact</a>
                    </li>
                </ul>
            </div>
        </nav>
    </div>

1. navbar-expand-sm

Voici une navbar de base avec un menu burger (.navbar-toggler) qui s'affiche sur un écran de petite taille (.navbar-expand-sm): sa résolution doit être au minimum de 576px.

2. navbar-expand-md et lien actif

Voici une navbar de base avec un menu burger (.navbar-toggler) qui s'affiche sur un écran de taille moyenne (.navbar-expand-md): sa résolution doit être au minimum de 768px.
Dans le cas 

<nav class="navbar navbar-expand-md navbar-light bg-light">

De plus, j'ai ajouté la classe .active sur le lien "Accueil" pour montrer que le lien actif sera mis en évidence et j'ai aussi ajouté l'attribut aria-current="page" pour les lecteurs d'écran. 

<a class="nav-link active" aria-current="page" href="#">Accueil</a>

Navbar
  • Accueil
  • Fonctionalité
  • Prix
  • Contact

Jouez sur la résolution pour constater que les différents points de menus se retrouveront dans le menu burger si la résolution minimale n'est pas atteinte.

De plus, j'ai ajouté le logo d'Eqla à gauche de la navbar. Pour cela, j'ai ajouté une balise img dans la balise a de la .navbar-brand.
