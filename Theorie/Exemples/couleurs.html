Les couleurs Bootstrap

Couleur de texte d'arrière-plan dans Bootstrap

Intro

Bootstrap utilise un ensemble de suffixes de couleurs bien connus. Ce sont toujours les mêmes. A nouveau pour rester cohérent.

Les voici: primary, secondary, success, danger, warning, info, light, dark, muted, white.

1. Le texte

Dans les exemples suivants, nous avons appliqué un .text-* à des balises p.

.text-primary, ce texte est bleu.

.text-secondary, ce texte est gris.

.text-success, ce texte est vert.

.text-danger, ce texte est rouge.

.text-warning, ce texte est orange.

.text-info, ce texte est bleu cyan.

.text-light, ce texte est très clair. On a jouté un fond noir .bg-dark pour le voir.

.text-dark, ce texte est gris noir.

.text-muted, ce texte est gris clair.

.text-white, ce texte est blanc. On a jouté un fond noir .bg-dark pour le voir.

3. L'arrière-plan/background dégradé

Dans les exemples suivants, nous avons appliqué un .bg-* et des .bg-gradient à des div.

Donc, comme pour les précédentes classes de Bootstrap, on peut chaîner des noms de classe dans l'attribut class de la balise HTML en cours.

.bg-primary .bg-gradient
.bg-secondary .bg-gradient
.bg-success .bg-gradient
.bg-danger .bg-gradient
.bg-warning .bg-gradient
.bg-info .bg-gradient
.bg-light .bg-gradient
.bg-dark .bg-gradient
